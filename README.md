✈️ Flight Delay Prediction System (End-to-End Machine Learning Project)

Developed an end-to-end machine learning system to predict flight arrival delays for Dallas/Fort Worth International Airport (DFW) using over 2.7M raw operational records enriched with historical weather data. Designed a two-stage modeling framework consisting of a binary classification model (delay vs. on-time) and a regression model to estimate delay duration in minutes. Implemented a comprehensive data pipeline involving ETD and CYCLE event filtering, reducing raw data to 1.12M unique flight-level records while eliminating duplicates and ensuring temporal consistency.

Engineered advanced features including temporal (hour, weekday, seasonality), operational (flight status updates, ETA volatility), and weather-based variables (wind speed, precipitation, visibility, pressure) using Open-Meteo API data. Applied feature selection techniques including multicollinearity reduction and leakage removal, followed by temporal train-test-validation split to simulate real-world forecasting conditions.

Trained and evaluated multiple machine learning models including Logistic Regression, Random Forest, XGBoost, and LightGBM. The optimized XGBoost classifier achieved 84% accuracy and 0.86 AUC-ROC, while the tuned LightGBM regressor achieved an R² of 0.76 with an average error of ~7.8 minutes. Performed hyperparameter tuning using RandomizedSearchCV and improved model generalization through threshold optimization. Enhanced interpretability using SHAP analysis to identify key drivers such as operational volatility, weather severity, and temporal patterns. The system enables proactive delay detection and duration forecasting to support airline scheduling, operational planning, and passenger communication.
